# 3D Chair Seat Selector

## Overview
A responsive and interactive seat selector built with HTML, CSS, and JavaScript. Seats are dynamically generated and styled as 3D chairs, allowing users to select available seats while booked or unavailable seats are disabled. The layout adapts to different screen sizes, with hover and selection effects for better interactivity.

## Features
- Dynamic seat generation from a JavaScript array  
- 3D chair visualization using CSS (`::before`, `::after`, legs divs)  
- Select and deselect available seats  
- Clear visual distinction for available, booked, and unavailable seats  
- Responsive layout using CSS Grid  
- Interactive hover and selection animations  
- Real-time display of selected seats  

## Technologies Used
- HTML5  
- CSS3 (3D styling, grid layout, animations)  
- JavaScript (DOM manipulation, event handling)
